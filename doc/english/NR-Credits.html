MUI

MUI, the work of Stefan Stuntz, is the most powerful UI object library available for the Amiga. Without it, this software would not have been possible. The MUI home page can be accessed via SASG. Please support these folks; MUI is responsible for a vast amount of Amiga software that would not otherwise exist. Every Amiga should have MUI installed.

The SASG home page is also one of the most well designed and visually pleasing pages on the net.
