Future Plans
We have many more plans for NewsRog. What's here now is just the beginning. Here are just a few of them.
More Filter Objects We have plans for many more interesting and useful selection and display filters. If you have ideas, please let us know!
External Control of Objects Eventually, all of NewsRog will be controllable via an external ARexx interface. This will let objects easily control other objects, and permit entire custom user interfaces for other programs to be created in NewsRog.
Message Catalogs Every object in NewsRog can be localized. Eventually, we plan to have non-English message catalogs for all of the program's classes.
Full ARexx Support A full ARexx command set to drive NewsRog from other applications.
Multipart Binaries Automatic detection and decoding of multipart binaries.
Integrated email support The ability to define email groups and use all the features of NewsRog as an emailer.
Programmer's Toolkit A toolkit and specifications for 3rd parties to create new classes for integration into NewsRog will be available.
PGP Support Verifying PGP signatures and adding PGP signatures to postings.
PPC Native Version If there is sufficient demand for native PPC Applications.
