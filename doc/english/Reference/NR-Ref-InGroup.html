Reference: Selection Filters: InGroup
Filter Overview This selection filter matches articles posted to the group currently being read. This filter is primarily for internal use by NewsRog; it's use externally is limited.

User Interface The user interface for this filter looks like this:

The following options are available:

  • First Group - If set, the filter will match only if the current group being read is the first group in the Newsgroups: header. Otherwise, it may be in any position.
