Reference: Selection Filters: SizeRange
Filter Overview This selection filter matches articles by size. You might, for example, filter against articles larger than 50 Kb, or those between 0 and 10 Kb.

Back to Top

User Interface The user interface for this filter looks like this:

The following options are available:

  • Min Size - The article must be at least this large in order to be selected by the filter.

  • Max Size - The article must be no larger than this in order to be be selected by the filter.

The size computation use the Bytes: header, which is usually, but not always, accurate.
