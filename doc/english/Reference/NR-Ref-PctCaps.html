Reference: Selection Filters: PctCaps
Filter Overview This selection filter matches all articles have more than a given percentage of capital letters in their subject. Articles with subjects written entirely in capital letters are rarely of value.

Back to Top

User Interface The user interface for this filter looks like this:

The available options are:

  • Min Length - Matched articles must have this minimum number of alphabetic characters in the subject field.

  • Cap % - This filter matches articles who's subject line contains at least this percentage of capital letters. Non alphabetic characters are not counted in the percentage.
