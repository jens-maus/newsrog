Reference: Selection Filters: DblSpace
Filter Overview This selection filter matches all articles which have more than a given number of doublespaced letters in their subject. For example:

    T h i s   I s   P r o b a b l y   S p a m !

Such articles are rarely of any value.

The available options are:

  • Min Length - Matched articles must have this minimum number of characters in the subject field. Articles with shorter subject fields are ignored.

  • Min DblSpace % - This filter matches articles who's subject line contains at least this percentage of doublespaced characters. Doublespace characters are characters followed by between one and four spaces.

  • DblSpace Chars - Characters in this set are detected as doublespace characters. For example, if an asterisk (*) is part of this string, then the subject "T*h*i*s i*s s*p*a*m" will be matched. .
