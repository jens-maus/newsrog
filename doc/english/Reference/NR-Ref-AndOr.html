Reference: Selection Filters: And/Or
Filter Overview This selection filter combines other filters using boolean operators. You might, for example, define a filter that matches any article that's posted by a particular person and is posted to less than three groups. Or you might define an anti-spam filter that matches articles with certain strings in their subject line or contains more than a certain number of lines.

User Interface The user interface for this filter looks like this:

The available controls are:

  • Add... - This allows you to add other filters to the list.

  • Edit... - This allows you to change parameters of filters that have been added to the list.

  • Delete - This deletes the selected filters from the list.

  • Clone - This makes a copy of the active filter.

  • Any/All - This changes the boolean mode. In Any mode, the And/Or filter will match an article if any of the filters in this list match (or mode). In All mode, the And/Or filter will match only if all filters in the list match (and mode).
