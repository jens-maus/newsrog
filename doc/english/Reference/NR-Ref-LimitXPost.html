Reference: Selection Filters: LimitXPost
Filter Overview This selection filter matches articles who are crossposted to a limited number of groups. This is useful for selecting against widely crossposted articles, which are rarely of value.

Back to Top

User Interface The user interface for this filter looks like this:

The following options are available:

  • Min Groups - The article must be crossposted to at least this many groups in order to be select by the filter.

  • Max Groups - The article must be crossposted to this many groups or less in order to be select by the filter.
