Reference: Selection Filters: LineRange
Filter Overview This selection filter matches articles that contain the indicated number of lines. You might, for example, filter against articles over 400 lines in a discussion group, or articles under 100 lines in a binaries group in order to remove non-binary posts.

User Interface The user interface for this filter looks like this:

The following options are available:

  • Min Lines - The article must contain at least this many lines in order to be selected by the filter.

  • Max Lines - The article must not contain more than this many lines in order to be selected by the filter.

The number of lines is obtained from the Lines: article header, which is usually, but not always, accurate.
