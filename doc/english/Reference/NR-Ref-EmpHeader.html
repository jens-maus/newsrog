Filter Overview This display filter emphasizes article header fields using a defined color and font style.

Back to Top

User Interface The user interface for this filter looks like this:

The available configuration options are:

  • Fold Lines - If selected, certain headers, such as Newsgroups: and References:, to have each of their entries displayed on a separate line. This avoids very long header lines for these fields.

  • Header Field Type Style - The defined font, color, and text style are used to emphasize the article header fields up to and including the separating colon character (":").

  • Header Data Type Style - The defined font, color, and text style are used to emphasize the article header after the separating colon character (":").
