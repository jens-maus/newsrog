Reference: Selection Filters: AgeRange
Filter Overview This selection filter matches articles by age. You might, for example, filter against articles between 2 and 4 days old.

Back to Top

User Interface The user interface for this filter looks like this:

The following options are available:

  • Min Age - The article must be at least this old in order to be selected by the filter.

  • Max Age - The article must be this old or younger in order to be selected by the filter.

The age computation takes into account the timezone of the posted article. Most, but not all articles have a correctly set timezone in their Date: header. Back to Top
