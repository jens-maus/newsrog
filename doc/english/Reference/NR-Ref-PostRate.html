Reference: Selection Filters: PostRate
Filter Overview This selection filter matches articles in threads with a certain rate of activity, as defined by the number of posts per 24 hours. The matching can be restricted to threads with no more (or less) than a certain number of posts. Matching is based on threads, so all articles in a given thread are matched identically.

Note that this filter works best as a selection filter, but not as a kill filter. Killing articles changes the number of articles in the thread and affects the measurement of this statistic in the future.

User Interface The user interface for this filter looks like this:

The following options are available:

  • Min Articles - The thread must have at least this many articles to be considered for selection.

  • Max Articles - The thread may have no more than this many articles to be considered for selection.

  • Min Per Day - As long as the thread meets the above two criteria, then any thread with at least this ratio of posts per 24-hour period...

  • Max Per Day - ... but not more than this ratio, will be selected.
