Reference: Selection Filters: MatchThreadBase
Filter Overview This selection filter matches all articles that are in a thread where the basenote of that thread matches a given condition, as defined by another selection filter. For example, you might define a filter to match articles that are in any thread where you posted the basenote.

There is only one control available:

  • The New... button adds a sub-filter. The MatchThreadBase filter will match any article in a thread where the basenote is matched by that filter.

After adding a sub-filter, that filter's user interface appears in the blank area and may be edited normally.
