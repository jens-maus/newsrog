Filter Overview This display filter spoiler protects parts of the article that are separated from the rest of the article by a form feed character (control-L). Multiple separate pages can be displayed.

Back to Top

User Interface The user interface for this filter looks like this:

The available configuration options are:

  • Format String - This string is displayed in the button that acts as the spoiler protector for a given page.

  • Font - If non-null, this font is used to display the text inside a spoiler page.

  • Color - If selected, this color is used to display the text in a spoiler page..

  • Style - The text style (bold, underline, or italic) for the displayed text inside a spoiler page.

Variables The following variables, in addition to the standard group, article, and project variables, are available for use in the spoiler Format String.
Variable NameDescription
MaxPage The maximum number of spoiler sections in this document.
Page The ordinal number of the current spoiler section.
Size The size in bytes of the current spoiler section.
Lines The number of lines in the current spoiler section.
