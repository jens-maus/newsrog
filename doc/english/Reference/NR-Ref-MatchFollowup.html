Reference: Selection Filters: MatchFollowup
Filter Overview This selection filter matches all articles that are followups to articles that match a given condition, as defined by another selection filter. For example, you might define a filter to match articles that are followups to articles you have posted. Or you might define a filter to match articles that are followups to articles posted to more than 5 groups, even if the followups themselves are not as widely crossposted.

User Interface The user interface for this filter looks like this:

The only control available is the New... button, which adds a sub-filter. The MatchFollowup filter will match any article that either is matched by the added filter, or is a followup to an article which is matched by that filter.

After adding a sub-filter, that filter's user interface appears in the blank area and may be changed normally.
