Reference: Selection Filters
And/Or This filter combines other filters using boolean operators.

Back to Top

AgeRange This filter selects articles by their current age in days, hours, minutes, and seconds. It compensates for the timezone of the article's author.

Back to Top

Cached This filter matches articles that have been cached on the local disk.

Back to Top

DblSpace This filter matches articles that have more than a given percentage of d o u b l e   s p a c e d characters in their subject field.

Back to Top

Header Match This filter matches one or more fixed strings or patterns in article headers.

Back to Top

In Group This filter matches articles in a particular group.

Back to Top

Limit XPost This filter matches articles crossposted to not less than a minimum and not more than a maximum number of groups.

Back to Top

Line Range This filter matches articles containing not less than a minimum and not more than a maximum number of body lines.

Back to Top

Marked This filter matches articles that have been marked in the group window.

Back to Top

Match Followup This filter matches all articles that are followups to articles that match a given condition. This lets you, for example, define a filter to show all followups to your own articles.

Back to Top

Match Thread This filter matches all articles that are in the same thread as articles that match a given condition. This lets you, for example, define a filter to show all articles in threads you have participated in.

Back to Top

Match Thread Base This filter matches all articles that are in a thread in which the basenote matches a given condition. This lets you, for example, define a filter to show all articles in threads that you started.

Back to Top

% Caps This filter matches all articles with subject lines containing less than a given percentage of capital letters.

Back to Top

New This filter matches all articles that are marked as newly downloaded. The new flag may be reset when articles are read, when the group is shut down, or both.

Back to Top

Permanent This filter matches all articles that have been marked as permanent in the group window.

Back to Top

PostRate This filter matches threads with a given rate of posting, as defined by the number of articles posted per day.

Back to Top

Read This filter matches all articles that have been read.

Back to Top

Repeat Char This filter matches all articles that have a given number of repeated characters in their subject line. This allows you to match articles containing, for example, "$$$$" or "!!!!". Such articles are rarely of value.

Back to Top

SizeRange This filter selects articles by their size in bytes, Kb, or Mb.

Back to Top

Subject Match This filter is a more limited version of the Header Match filter. Unlike Header Match, it does not allow matching against more than one pattern.

Back to Top

Thread Base This filter matches articles that are the basenote of a thread.
