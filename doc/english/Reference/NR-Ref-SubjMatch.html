Reference: Selection Filters: SubjMatch
Filter Overview This selection filter matches articles that contain a particular subject string exactly. The Header Match filter is more flexible and generally useful in most cases.

Back to Top

User Interface The user interface for this filter looks like this:

The following options are available:

  • String - The subject string to match.

  • Match Case - If set, the match is case sensitive.

  • SubString - If set, the given string may be a substring of the article's subject. Otherwise, it must match the article subject exactly.
