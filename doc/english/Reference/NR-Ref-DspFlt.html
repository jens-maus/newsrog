Reference: Display Filters
EMail This filter creates clickable email addresses and message-IDs.

Back to Top

CharMap This filter creates display-time character table mappings.

Back to Top

Emphasize Headers This filter emphasizes header fields & data in a definable color and font style.

Back to Top

Hide Headers This filter hides various unwanted header fields.

Back to Top

Hide Sig This filter replaces signature blocks with a small click box that can be clicked to view the entire signature.

Back to Top

Highlight Text Patterns This filter highlights article text matching certain patterns using a defined color and font. For example, you might highlight your cat's name appearing in article bodies in the rec.pets.cats.* hierarchy, or highlight common acronyms such as ROTFL or FWIW.

Back to Top

Highlight & Trim Quotes This filter highlights quoted text in user definable color and font styles. It also trims large quoted text blocks to smaller scrollable or clickable regions in order to avoid seeing overquoted text.

Back to Top

MIME Decoder This filter decodes MIME messages. It can display inline images, decode base64, quoted-printable, x-uue/x-uuencode, and handle multipart mime messages.

Back to Top

Rewrap This filter rewraps short or long text lines to a given percentage of the display width.

Back to Top

ROT13 This filter performs an ROT13 operation on the article body.

Back to Top

Scroll Headers This filter puts article header lines in a scrolling block, and rewraps long header lines.

Back to Top

Spoilers This filter adds spoiler protection to articles which use a formfeed as the spoiler character.

Back to Top

Substitute This filter substitutes user defined strings or small graphic images for patterns in the displayed article.

Back to Top

Text Styles This filter converts text between asterisks to bold, text between underscore characters to underline, and text between forward slashes to italic.

Back to Top

Trim Space This filter trims repeated blank lines in an article body.

Back to Top

URL This filter creates clickable URL's.

Back to Top

UUDecode This filter handles uuencoded data in article bodies, with the ability to display inline image data for which an AmigaOS Datatype is installed.
