Reference: Selection Filters: MatchThread
Filter Overview This selection filter matches all articles that are in the same thread as articles that match a given condition, as defined by another selection filter. For example, you might define a filter to match articles that are in any thread you have posted to, whether those articles are followups to yours or not.

User Interface The user interface for this filter looks like this:

There are two controls available:

  • Mode - This changes the matching mode between Any, in which all articles in a thread are selected if any article in the thread matches the sub-filter, and All, in which articles are only selected if all other articles in the thread match.

  • The New... button adds a sub-filter. The MatchThread filter will match any article that either is matched by the added filter, or is in the same thread as any article matched by that filter.

After adding a sub-filter, that filter's user interface appears in the blank area and may be edited normally.
