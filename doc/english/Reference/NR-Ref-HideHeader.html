Filter Overview This display filter hides selected header fields that you don't wish to display with every article.

User Interface The user interface for this filter looks like this:

The available configuration options are:

  • Header Patterns - Only headers matching one or more of these AmigaOS patterns will be displayed. All others will be hidden. Headers will be displayed in the order they are matched by entries in this list, so that you can change the display order of the headers (for example, displaying From: before Subject:, or vice versa) by drag-sorting this list. If a header is matched by more than one pattern in this list, it will only be displayed once, in the position dictated by the first match.

    The patterns are not case sensitive, but must match the trailing colon (":") character in the header field.
