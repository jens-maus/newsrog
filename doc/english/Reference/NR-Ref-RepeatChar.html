Reference: Selection Filters: RepeatChar
Filter Overview This selection filter matches articles that contain more than a given number of repeated characters in their subject. For example, you might match articles containing strings such as "$$$$" or "$!$!$!".

User Interface The user interface for this filter looks like this:

The following options are available:

  • Max Count - Articles with at least this many repeated characters from the given set are matched.

  • Characters - Only characters in this set are counted. Other characters may repeat any number of times.

  • Set - If this options is selected, the characters are treated as a set. In this case, if Max Count is 4 and Characters contains !?, then FOO!?!? would match. If unselected, FOO!!!! or FOO???? would match but FOO!?!?! would not.
