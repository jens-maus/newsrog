Tutorial 4: Initial Configuration

Default Project When you first install NewsRog, a default project is created that contains several newsgroups and some display filters. You can add or remove from this default project as you wish, but there are several items that must be configured to properly use the program.

Step 1:
Adding a Server 		In order to read news, you need to configure at least one news server. To do this, enter the configuration area by pressing the Servers button. This will bring up the server list editor. Here you can define any number of NNTP servers. If you have previously been using tin or another newsreader that supports the NNTPSERVER environment variable, then you can use the Get Env button to import a news server from the NNTPSERVER variable. Otherwise, add a new news server by clicking on the Add Server button and typing in the name of your news server. Most ISPs provide news service to their customers. Your ISP should have provided you with the name of your news server when you registered for internet service. It is possible that just using "news" will work here if you have a properly set up domain list in your internet software and your news server is named news.yourdomain.com. Consult the documentation for your internet software for more information on domain lists.

No further configuration of the news server is necessary.

Step 2:
User Config 		The 2nd and last configuration step necessary is to configure your email address. Enter the configuration editor and select the Program Options tab (the second from the left). If your name is not already set in the Real Name box, you can enter it now. Also enter your email address in the appropriate box. The Organization field can be set to any short string and will be displayed when other people read articles that you post. You may also adjust other options in this window, but for now it is best to leave the default configuration alone.
